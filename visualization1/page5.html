The previous page focuses on estimating $\phi_0$ by minimizing the innovations dependencies. This page shows that minimizing the innovations dependencies is equivalent to maximizing the shocks non-Gaussianity.i Consequently, instead of trying to find the least independent innovations, we can also find the most non-Gaussian innovations.

The non-Gaussianity of the innovations $ e_t(B(\phi)) $ can be measured using the skewness and excess kurtosis: $$ \begin{matrix} \text{Skewness} & E[e_{1,t}(B)^3] \\ \text{Skewness} & E[e_{2,t}(B)^3] \\ \text{Excess kurtosis} & E[e_{1,t}(B)^4-3] \\ \text{Excess kurtosis} & E[e_{2,t}(B)^4-3] \end{matrix} $$

Move the $\phi$ slider to see how the rotation angle affects the on-Gaussianity of the innovations displayed in the table below the scatter plot.

The loss value in the table above simply summes up all squared higher-order moments to get an overall measure of the non-Gaussianity $$ loss(\phi) = mean( e_{1,t}(\phi)^3 )^2 + mean( e_{2,t}(\phi)^3 )^2 + mean( e_{1,t}(\phi)^4-3 )^2 + mean( e_{2,t}(\phi)^4-3 )^2 $$ The plot below shows how the loss for all rotation angles $\phi$.
Push the Maximize-Non-Gaussianity-Button to find the rotation angle $\phi$ which leads to the most non-Gaussian innovations.
Try out different sample sizes $T$ and new data sets using the New-Data-Button to see how the rotation $\phi$ which maxiizes the innovations' non-Gaussianity is located arround the true rotation $\phi_0=0.5$.

Take-away: Choosing the angle $\phi$ which minimizes the innovations' non-Gaussianity allows to estimate $\phi_0$.i
